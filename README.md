# Yared Girma — Portfolio

A modern, dark editorial portfolio website built with React + Vite.

## Features
- Custom animated cursor with trail effect
- Asymmetric editorial layout (Fraunces × Syne × DM Mono)
- Scroll-reveal animations
- Parallax photo effect
- Animated skill bars
- Copy-to-clipboard contact
- Fully responsive

## Setup

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

Then open `http://localhost:5173` in your browser.

## Tech Stack
- React 18
- Vite 5
- CSS Modules
- Google Fonts (Fraunces, Syne, DM Mono)
